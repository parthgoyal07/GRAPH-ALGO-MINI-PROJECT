# Graph Algorithms – Assignment 3

## 📌 Overview

Implementation of graph algorithms for traversal, shortest path, and spanning trees.

## 🧠 Concepts Covered

* Graph Representation (Adjacency List & Matrix)
* BFS & DFS
* Topological Sorting
* Dijkstra & Bellman-Ford
* Minimum Spanning Tree (Prim)

## ⚙️ Implementation

Tasks:

1. Graph representation
2. BFS and DFS traversal
3. Topological sorting (DAG)
4. Shortest path algorithms
5. MST computation

## 📊 Results

* BFS/DFS explore graphs efficiently
* Dijkstra faster but no negative edges
* Bellman-Ford handles negatives
* MST minimizes cost

## 📚 Conclusion

Graph algorithms are essential in real-world networks.
